# How to add Follow Gate

Follow Gate sends your link only after the user follows you. Non-followers see a "follow to unlock" message instead.

## Step 1: Open the automation

Open the Instant Automation you want to gate.

## Step 2: Find Options

Scroll to the **Options** section.

![Options section in automation editor](/help-screenshots/automation-options-section.png)

## Step 3: Toggle Follow Gate

Turn **Follow Gate** on.

![Follow Gate toggle on](/help-screenshots/follow-gate-toggle-on.png)

## Step 4: Customize the gate message

Edit the "follow us to unlock" message. Default copy works, but a personal line converts better.

![Follow Gate message customization field](/help-screenshots/follow-gate-message-edit.png)

## Step 5: Save and activate

Toggle the automation **Active**. The gate runs on every new comment.

## How it works

1. User comments your keyword
2. Botyoo checks if they follow you
3. **Following** → original DM is sent immediately
4. **Not following** → Botyoo sends the gate message ("follow to unlock"). Once they follow, the original DM is sent automatically

## When to use it

- Discount drops where you want the audience growth
- Lead magnets where the freebie is the incentive to follow
- Any one-off offer where the trade ("follow, then get the link") feels fair
