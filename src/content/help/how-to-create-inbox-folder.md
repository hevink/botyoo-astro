# How to create an Inbox folder

Folders are reusable buckets for grouping contacts. They show up in the Inbox sidebar, as a destination in flow steps, and as a filter in campaign audiences.

## Step 1: Open Inbox

Click **Inbox** in the sidebar.

## Step 2: Click "Create folder"

The **Folders** section at the top of the left rail has a **+ Create folder** action.

![Folders section in Inbox with Create folder option](/help-screenshots/inbox-folders-section.png)

## Step 3: Name the folder

Type a name (e.g. `Supporters`, `VIP`, `Hot leads`). Press Enter.

![Create folder input with name being typed](/help-screenshots/inbox-create-folder-input.png)

## Step 4: Add conversations

There are three ways to add contacts to a folder:

### From a conversation
Open any conversation. In the header, click **Add to folder** and pick the folder.

![Add to folder action in conversation header](/help-screenshots/inbox-add-to-folder.png)

### From a flow
Use the **Add contact to folder(s)** step inside any flow. Pick the folder.

![Add to folder step in flow builder](/help-screenshots/flow-add-to-folder-step.png)

### From a contact in Contacts
The Contacts page links each contact to its tags and folders.

## Step 5: Use the folder

Once a folder has contacts, you can:

- Filter the Inbox by clicking the folder in the left rail
- Target a Campaign audience by folder
- Reference the folder in conditional logic (rolling out)
