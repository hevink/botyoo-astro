# How to redeem your RocketHub Lifetime Deal code

Bought a Botyoo Lifetime Deal on RocketHub? Your code activates the **plan you purchased for life**, no recurring payment, no card on file. Here's how to apply it.

## Before you start

- Make sure you've created your Botyoo account using the same email you'd like to use the LTD on. [Sign up here](/help/how-to-sign-up) if you haven't.
- Keep your RocketHub redemption code handy. It looks like `RHBT-XXX-XXXXXXXXXXXX` and is on your RocketHub purchase page.

## Step 1: Open Settings

Click **Settings** at the bottom of the left sidebar.

![Settings option highlighted in the Botyoo sidebar](/help-screenshots/rockethub-settings-sidebar.png)

## Step 2: Switch to the Plan & Billing tab

At the top of the Settings page, click the **Plan & Billing** tab.

![Plan & Billing tab highlighted in Settings](/help-screenshots/rockethub-plan-billing-tab.png)

## Step 3: Enter your RocketHub code

Scroll down to the **Redeem your RocketHub Deal** card. Paste your code into the input field, then click **Redeem**.

![Redeem your RocketHub Deal card with code input field](/help-screenshots/rockethub-redeem-section.png)

No payment or card details are required, the code itself activates the plan.

## Step 4: Your lifetime plan is active

Once the code is accepted, your workspace flips to the **lifetime plan you purchased on RocketHub**. You'll see the **Lifetime Deal** badge and confirmation that there's nothing more to pay, ever.

![Lifetime plan confirmation screen](/help-screenshots/rockethub-success-agency-plan.png)

The exact plan tier and limits depend on the deal you bought, refer to your RocketHub purchase page for what's included.

## Troubleshooting

**"Invalid code" error.** Double-check the code, codes are case-insensitive but the dashes matter. Copy directly from your RocketHub purchase page to avoid typos.

**"Code already redeemed" error.** Each code can only be used once and is tied to the first workspace it's redeemed on. If you've already applied it, your Agency plan is active on that workspace, switch into it from the workspace switcher in the top-left.

**Need to stack codes for higher tiers?** RocketHub LTD tiers stack workspace and team-member limits. Redeem each code one after another from the same screen on the same account.

If anything looks off after redemption, open the **💬 chat widget** in the bottom-right corner or email [support@botyoo.com](mailto:support@botyoo.com), include your redemption code and we'll sort it out.
