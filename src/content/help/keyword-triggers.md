# Keyword Triggers

Keyword triggers fire your automation when a comment or DM contains a specific word or phrase.

## Setting Up a Keyword Trigger

1. Create or edit an automation
2. Select **Comment contains keyword** or **DM contains keyword** as your trigger
3. Enter your keyword(s) in the trigger settings
4. Save and activate

## Keyword Matching Options

| Option | Behavior |
|---|---|
| Exact match | Only triggers if the comment/message is exactly the keyword |
| Contains | Triggers if the keyword appears anywhere in the message |
| Case sensitive | Toggles case sensitivity (off by default) |

## Multiple Keywords

You can add multiple keywords to a single trigger — the automation fires if **any** of the keywords match. Add them comma-separated or one per line depending on the input field.

## Tips

- Use short, common words like "price", "link", or "info" that users are likely to type naturally
- Avoid overly generic words that might trigger unintended automations
- Test your trigger by commenting on a post yourself before going live
